# Gnome

### Gnome — это мобильная 2D игра-платформер на Unity.

Разработка игры прекращена.

Вы можете использовать эту игру в качестве шаблона или заимствовать некоторые её механики.

#### В Gnome реализованы несколько стандартных механик:
- Разные типы управления: джойстик, кнопки, джойстик + кнопки.
- Настройки FPS и способа передвижения.
